About Me

Get to know more about my background and expertise

Building scalable, reliable, and impactful applications

Hello! I'm Charles, a Full-Stack Software Engineer with a strong backend focus. I specialize in architecting and building high-performance systems using Node.js (Express/NestJS), PHP (Laravel), and Python (Flask/FastAPI), with expertise in relational and non-relational databases like MySQL, PostgreSQL, and MongoDB.

On the frontend, I craft responsive, user-friendly interfaces with React, TypeScript, TailwindCSS, and Bootstrap, ensuring seamless integration with backend services.

I have hands-on experience with cloud platforms (AWS, DigitalOcean), containerization (Docker, Docker Compose), and caching/messaging systems (Redis, RabbitMQ) to deliver scalable and production-ready solutions.

When I’m not coding, I explore new system design approaches, contribute to open-source projects, or experiment with automation tools that make developers’ lives easier.
